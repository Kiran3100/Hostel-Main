# Complete Utility Classes Overview

## 1. `constants.py` - System Constants
**No classes** - Contains constants, enums, and configuration variables:
- User roles, statuses, choices
- File limits, validation patterns
- Error/success messages
- Feature flags, system limits

## 2. `datetime_utils.py` - Date/Time Operations

| Class | Purpose |
|-------|---------|
| `DateTimeHelper` | Basic date/time operations (now, parse, format, duration calculations) |
| `TimezoneConverter` | Convert between timezones, UTC handling |
| `DateRangeCalculator` | Generate date ranges, calculate overlaps, get month/quarter ranges |
| `BusinessHoursChecker` | Check business hours, calculate business days, handle holidays |
| `AgeCalculator` | Calculate age, check if minor, age categories, next birthday |
| `ScheduleHelper` | Generate recurring dates, next occurrences, check if due |
| `AttendanceDateHelper` | Attendance-specific date calculations, semester dates |
| `LeaveCalculator` | Calculate leave days, validate leave periods |
| `DateTimeValidator` | Validate date strings, check future/past dates |
| `DateTimeFormatter` | Advanced date/time formatting, relative dates |

## Summary by Category

**Text Processing:** StringHelper, TextCleaner, NameFormatter, SlugHelper
**Date/Time:** DateTimeHelper, TimezoneConverter, BusinessHoursChecker
**Files:** FileHelper, ImageProcessor, PDFGenerator, ExcelGenerator
**Communication:** EmailHelper, SMSHelper
**Security:** EncryptionHelper, PasswordHasher, SecurityValidator
**Data:** DataValidator, BusinessRuleValidator, PaginationHelper
**Location:** GeoLocationHelper, AddressGeocoder
**Formatting:** All formatter classes for display/export

Each class is focused on a specific functionality with clear separation of concerns.

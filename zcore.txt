# Classes in Core Module Files

## Dependencies (dependencies.py)
- **DatabaseDependency**: Provides database session dependency
- **AuthenticationDependency**: Handles user authentication from JWT tokens
- **AuthorizationDependency**: Manages permission-based authorization
- **TenantContextDependency**: Extracts multi-tenant context
- **CurrentUserDependency**: Loads user with role relationships
- **SuperAdminDependency**: Super admin access control
- **AdminDependency**: Admin level access control
- **SupervisorDependency**: Supervisor level access control
- **StudentDependency**: Student access control
- **HostelContextDependency**: Hostel context validation
- **PaginationDependency**: Pagination parameter handling
- **FilterDependency**: Query filtering
- **CacheDependency**: Cache service provider
- **NotificationDependency**: Notification service provider
- **FileUploadDependency**: File upload validation

## Exceptions (exceptions.py)
- **BaseHostelException**: Base exception class
- **AuthenticationException**: Authentication failures
- **AuthorizationException**: Permission/access failures
- **ValidationException**: Data validation errors
- **BusinessLogicException**: Business rule violations
- **DatabaseException**: Database operation failures
- **ExternalServiceException**: External service failures
- **PaymentException**: Payment processing errors
- **BookingException**: Booking operation errors
- **RoomUnavailableException**: Room availability errors
- **InsufficientPermissionsException**: Permission errors
- **TokenExpiredException**: Token expiration errors
- **InvalidCredentialsException**: Invalid credential errors
- **TenantNotFoundException**: Tenant not found errors
- **HostelNotFoundException**: Hostel not found errors
- **StudentNotFoundException**: Student not found errors
- **DuplicateResourceException**: Resource already exists errors
- **FileUploadException**: File upload errors
- **NotificationException**: Notification delivery errors
- **MaintenanceException**: Maintenance operation errors
- **ComplaintException**: Complaint handling errors

## Domain Events (domain_events.py)
- **EventCategory**: Event category enum
- **EventSeverity**: Event severity enum
- **BaseDomainEvent**: Base event class
- Various domain-specific events:
  - User events (UserRegisteredEvent, UserVerifiedEvent, etc.)
  - Booking events (BookingCreatedEvent, BookingConfirmedEvent, etc.)
  - Payment events (PaymentProcessedEvent, PaymentFailedEvent, etc.)
  - Complaint events (ComplaintCreatedEvent, ComplaintResolvedEvent, etc.)
  - Maintenance events
  - Student events
  - Room events
  - Attendance events
  - Announcement events
  - And more...
